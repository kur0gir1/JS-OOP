Definitions

Named Functions

Are traditional functions declared with the function keyword followed by the name of the function

Example:
function add(a,b){
  return a + b;
}
console.log(add(1,3));

Arrow Functions

Arrow functions allow you to write functions with less code. They are defined using the arrow (=>) syntax and do not have their own this context.
Arrow functions are not suitable as object methods where you need a dynamic this, nor can they be used as constructors

Example:
const subtract = (a,b) =>{
  return a - b;
}
console.log(subtract(5,2));

Async Functions

Are functions declared with the async keyword followed by the name of the function
Async functions always return a promise, and the await keyword is used to wait for the promise to resolve before continuing with the rest of the code.

Example:
async function fetchData(){
  const data = await fetch('https://jsonplaceholder.typicode.com/posts/1');
  return data.json();
}
console.log(fetchData());
